
## Aprendizaje esperado 7
El Proyecto de Aprendizaje esperado N° 7 está diseñado para desarrollar las habilidades esenciales para trabajar con herramientas de repositorio como Git, así como herramientas de repositorio remoto como GitHub.

Se implementaron funciones de Git, tales como iniciar un Git, creando un repositorio, también se usaron herramientas de trabajo colaborativo a través de repositrios remotos de equipo, aplicando las técnicas de envío y recepción de actualizaciones de progreso en diversas ramas creadas y su posterior integración para la creación del resultado sólido. 

Las ramas utilizadas fueron:
- Main
- Feature/Navbar
- Feature/Footer
